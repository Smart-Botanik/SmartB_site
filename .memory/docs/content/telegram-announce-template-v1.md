# Шаблон анонса статьи в Telegram (CONTENT-LCH-2 / CONTENT-TG-1)

**Канал:** SmartБотаник  
**Использование:** поле `bodyTelegramMd` у гайда; кнопка «Отправить в Telegram» в admin.

---

## Описание канала (для t.me / about)

```
SmartБотаник — гайды по выращиванию и пост-урожаю.

Короткие советы, анонсы статей и ссылки на полные материалы:
https://smart-botanik.ru/guides

Приложение для дневника сада — скоро.
```

## Pinned (закрепить первым постом)

```
👋 Добро пожаловать в SmartБотаник

Здесь публикуем:
• короткие советы по культурам и среде
• анонсы гайдов с сайта
• новости продукта (когда появится приложение)

Полные статьи — на сайте:
https://smart-botanik.ru/guides

Подписывайтесь и сохраняйте полезное 🌱
```

## Шаблон анонса гайда (`bodyTelegramMd`)

```markdown
🌱 {заголовок}

{2–4 предложения сути}

• пункт 1
• пункт 2
• пункт 3

📖 Полная статья: https://smart-botanik.ru/guides/{slug}
```

Hashtags добавляет backend (BK-TG-2) в footer при publish — в MD дублировать не обязательно.
